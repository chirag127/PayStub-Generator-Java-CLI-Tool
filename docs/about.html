Employee-slip-app-BM-JWDC is a simple command-line application developed using Java. It takes input from the user such as employee name, ID, and basic salary, and calculates various components of the salary slip including HRA, DA, TA, PF, and Gross Salary.

Inputs:

  • Employee Name
  • Employee ID
  • Basic Salary

Outputs:

  • HRA, DA, TA
  • PF (Provident Fund)
  • Gross Salary
  • Formatted Salary Slip
