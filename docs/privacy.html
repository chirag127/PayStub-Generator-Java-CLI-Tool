Privacy Policy

Last Updated: June 5, 2026

Welcome to Employee-slip-app-BM-JWDC, a course project developed as part of the Java Web Development Course by Brain Mentors. This Privacy Policy explains how we handle any information collected through the use of this application.

1. Information We Collect

The Employee-slip-app-BM-JWDC is a command-line interface (CLI) application that runs locally on your device. It does not collect, store, transmit, or process any personal data on external servers. The only data processed are:

  • Employee Name
  • Employee ID
  • Basic Salary

These inputs are used solely to generate a salary slip for display or local saving. All data remains on the user's local machine and is not shared with any third party.

2. Data Usage

Any information entered into the application is used in real-time to calculate and display:

  • HRA (House Rent Allowance)
  • DA (Dearness Allowance)
  • TA (Travel Allowance)
  • PF (Provident Fund)
  • Gross Salary

No data is logged, persisted beyond the session, or used for any other purpose.

3. Data Security

Since no data is transmitted over the internet or stored remotely, the application inherently respects user privacy. Users are responsible for securing their local environment where the application is executed.

4. No Third-Party Sharing

We do not share any user data with third parties. The application has no connectivity to external databases, APIs, or tracking services.

5. Children's Privacy

This application is not directed toward individuals under the age of 16, and no data is knowingly collected from such individuals.

6. Changes to This Policy

As this is a static educational project, this privacy policy is not expected to change. However, if updates occur, they will be posted here with a revised "Last Updated" date.
