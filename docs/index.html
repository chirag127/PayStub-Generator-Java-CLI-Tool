Employee Salary Slip Generator

A simple Java CLI application developed as part of the Brain Mentors Java Web Development Course. It takes employee details and generates a detailed salary slip with allowances and deductions.

  • Employee Name ‚Äì Full name of the employee

  • Basic Salary ‚Äì Base salary amount (INR)

üìÑ Generated Output

  • TA (15%) ‚Äì Travel Allowance
  • PF (12%) ‚Äì Provident Fund Deduction
  • Gross Salary ‚Äì Total earnings after deductions

This command-line interface (CLI) application is fully written in Java, demonstrating core programming concepts like input handling, arithmetic operations, and formatted output. 

public class SalarySlip {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter Employee Name: ");
        String name = sc.nextLine();
        
        System.out.print("Enter Employee ID: ");
        int id = sc.nextInt();
        
        System.out.print("Enter Basic Salary: ");
        double basic = sc.nextDouble();

        double hra = 0.5 * basic;
        double da = 0.3 * basic;
        double ta = 0.15 * basic;
        double pf = 0.12 * basic;
        double gross = basic + hra + da + ta - pf;

        System.out.println("\n--- Salary Slip ---");
        System.out.println("Name: " + name);
        System.out.println("ID: " + id);
        System.out.println("Basic: " + basic);
        System.out.println("HRA: " + hra);
        System.out.println("DA: " + da);
        System.out.println("TA: " + ta);
        System.out.println("PF: " + pf);
        System.out.println("Gross: " + gross);
    }
}
